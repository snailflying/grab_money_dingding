
##钉钉红包插件介绍

这个Android插件可以帮助你在钉钉群聊抢红包时战无不胜。当检测到红包时，插件会自动点击屏幕，人工点击的速度无法比拟。

这个版本是初始版本。不可避免的存在bug。如果发现问题，请联系我


## 最新版本

1. 服务运行时防止息屏。
2.增加状态栏监听

## 使用方法

1. 打开『钉钉红包』应用，开启插件。
2. 坐等红包进账。

## 实现原理

利用AccessibilityService实现红包辅助。
